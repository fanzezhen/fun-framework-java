fun-framework-security-sa-token
------------------------------------------
错误码格式 151**

Sa-Token 安全组件集成

# 功能介绍

基于 Sa-Token 实现的轻量级安全认证：
- 登录认证
- 权限验证
- 角色管理
- Session 管理

# 快速开始

## 一、添加依赖

在 pom.xml 中添加以下依赖：

    <dependency>
        <groupId>com.github.fanzezhen</groupId>
        <artifactId>fun-framework-security-sa-token</artifactId>
    </dependency>

## 二、配置

在 application.yml 中配置 Sa-Token 相关参数
