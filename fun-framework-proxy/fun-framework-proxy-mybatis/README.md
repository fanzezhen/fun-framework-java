fun-framework-proxy-mybatis
------------------------------------------
错误码格式 182**

基于 MyBatis 的静态资源代理组件

# 功能介绍

在 MyBatis 实体类字段上实现代理功能

# 快速开始

## 一、添加依赖

在 pom.xml 中添加以下依赖：

    <dependency>
        <groupId>com.github.fanzezhen</groupId>
        <artifactId>fun-framework-proxy-mybatis</artifactId>
    </dependency>

## 二、使用注解

在 MyBatis 实体类的字段上标注@ProxyField
