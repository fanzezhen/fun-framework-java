fun-framework-proxy-orika
------------------------------------------
错误码格式 183**

基于 Orika 的静态资源代理组件

# 功能介绍

在使用 Orika 进行对象复制时实现字段代理功能

# 快速开始

## 一、添加依赖

在 pom.xml 中添加以下依赖：

    <dependency>
        <groupId>com.github.fanzezhen</groupId>
        <artifactId>fun-framework-proxy-orika</artifactId>
    </dependency>

## 二、配置

需要在 yml 中启用 orika 代理配置
