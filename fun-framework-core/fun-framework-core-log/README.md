日志组件
------------------------------------------
错误码格式 10***

提供日志记录相关功能，包括基础日志和 Web 日志

# 子模块说明

| 模块名 | 描述 | 错误码格式 |
|--------|------|-----------|
| [fun-framework-core-log-base](fun-framework-core-log-base) | 日志基础组件，提供 traceId 生成和日志配置 | 90600 |
| [fun-framework-core-log-web](fun-framework-core-log-web) | Web 日志组件，提供 Web 请求日志过滤 | 90601 |

# 配置说明

详细配置请参考各子模块 README
